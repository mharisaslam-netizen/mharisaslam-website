# mharisaslam.com

Static, crawl-ready personal website for Muhammad Haris Aslam.

## Local validation

```text
npm run build
npm run check
```

The build writes production files to `dist/`. Vercel serves that directory, applies clean URLs, and redirects the former `.dc.html` paths to their canonical replacements.

## Content principles

- Use only verified experience already documented by Muhammad Haris Aslam.
- Do not claim physical offices or legal entities for market coverage.
- Do not publish placeholder article URLs; add an insight only when its complete body and evidence are ready.
- Keep one consistent Person entity at `https://www.mharisaslam.com/#person`.
